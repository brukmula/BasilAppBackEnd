# Basil App Backend Code
# Install the dependencies:
## Firebase CLI
See the instructions [here](https://firebase.google.com/docs/cli#install_the_firebase_cli)
## npm packages
```sh
npm install
```
# Run the app:
```shell
DEBUG=BasilAppBackEnd:* npm start
```

**or**

```shell
node app.js
```
